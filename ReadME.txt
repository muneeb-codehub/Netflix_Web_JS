This project is a Netflix front page clone built using HTML, CSS, and JavaScript. It includes a Sign-In and Sign-Up page, dynamically generated with JavaScript, along with full form validation.

->Features
Netflix-style home page with hero section and email signup
Fully responsive design
Sign-In Page dynamically generated using JavaScript
Sign-Up Page with form validation and plan selection
Custom JavaScript form validation for user input
Background image and dark overlay effect

->Technologies Used
HTML → Structure of the pages
CSS → Styling & layout
JavaScript (DOM Manipulation) → Dynamic content generation

->Project Structure
/Netflix-Muneeb
│── index.html        # Home page
│── signin.html       # Sign-In page 
│── signup.html       # Sign-Up page 
│── style.css         # Main CSS file
│── script.js         # JavaScript file for functionality
│── images/           # Folder containing background and logo images
└── README.md         # Project documentation

->How It Works
JavaScript dynamically creates the Sign-In & Sign-Up pages.
Forms include validation (email, password strength, etc.).
Dropdown for plan selection in the Sign-Up page.
