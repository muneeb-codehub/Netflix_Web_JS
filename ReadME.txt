This project is a Netflix front page clone built using HTML, CSS, and JavaScript. It includes a Sign-In and Sign-Up page, dynamically generated with JavaScript, along with full form validation.

->Features
Netflix-style home page with hero section and email signup
Fully responsive design
Sign-In Page dynamically generated using JavaScript
Sign-Up Page with form validation and plan selection
Custom JavaScript form validation for user input
Background image and dark overlay effect
